# forMe 🍃

**forMe** is a cozy, matcha-themed personal productivity application designed to create a calming space for focused work and intentional planning. 🕰️🍵✨

> [!IMPORTANT]
> **Personal Project Notice**: This is a personal project built specifically for my own productivity and study needs. It is tailored to my unique workflow and aesthetic preferences.

---

## 📟 Features

### 🍵 Digital Alarm (Focus Mode)
A premium, rectangular digital alarm clock interface with "glassmorphism" depth.
- **Tactile Controls**: Play/Pause, Reset, +1m/-1m adjustments.
- **Quick Shift**: Dedicated buttons for +5m and -5m focus extensions.
- **Dynamic LEDs**: Pulsing status indicators that match the current theme (Matcha Green or Soft Pink).

### 🗓️ Visual Timetable Planner
A full-calendar system for routine design.
- **Monthly View**: Soft-rounded day cards with interactive hover effects and activity indicators.
- **Vertical Timeline**: Plan your day from **8:00 AM to 11:00 PM** with aesthetic task blocks.
- **Persistence**: All plans are synced directly to MongoDB for reliable access.

### 📊 Dashboard Shortcut
My homepage roadmap.
- **Today's Roadmap**: An instant view of your planned tasks for the current day.
- **Analytics**: Live tracking of Focus Hours, Completed Sessions, Consistency, and Streaks.

### 🗓️ Routine Builder & Study Journal
- **Scheduler**: Design repeating weekly routines with ease.
- **Daily Log**: A minimal journal to document breakthroughs and study reflections.

---

## 🛠️ Tech Stack

- **Frontend**: React (Vite), Zustand (State Management), Framer Motion (Animations), Lucide-React (Icons).
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB Atlas (Mongoose).
- **Styling**: Vanilla CSS with a focus on Glassmorphism and specialized Figma-to-Code aesthetics.

---

## 🚀 Deployment

- **Frontend**: Hosted on **Vercel**.
- **Backend**: Hosted on **Render**.
- **Storage**: **MongoDB Atlas**.
